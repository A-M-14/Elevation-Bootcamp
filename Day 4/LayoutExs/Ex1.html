Welcome to the Holy Grail Layout

This is a demonstration of the classic "Holy Grail" layout pattern implemented using CSS Flexbox. The Holy Grail layout is a web page layout that consists of a header, footer, and three columns. The center column contains the main content, while the left and right columns contain supplementary content like navigation or advertisements.

Key Features

This layout demonstrates several important web development concepts including flexible content areas, sticky footer positioning, and responsive design principles that adapt to different screen sizes.

You can add more content here to test the sticky footer behavior. When there's enough content to fill the viewport, the footer will stay at the bottom and you'll need to scroll to see it. When there's minimal content, the footer will stick to the bottom of the viewport.
